# 🎬 MovieVerse

A modern movie discovery app built using:

- Node.js
- Express.js
- EJS
- TMDB API
- OMDB API

## Features

- 🔍 Movie Search
- ⭐ Hover Preview
- 🎬 Trending Movies
- 🆕 Latest Movies
- 🏆 Top Rated Movies
- 🎭 Genre Filter
- 📱 Responsive UI

## Installation

Clone the repo:

git clone https://github.com/Jeffrey-29/movie-verse.git

Install dependencies:

npm install

Create .env file:

TMDB_TOKEN=your_tmdb_token
OMDB_API_KEY=your_omdb_key

Run:

nodemon index.js

Open:

http://localhost:3000
